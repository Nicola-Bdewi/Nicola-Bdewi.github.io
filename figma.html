Figma Page

Our sign up page will ask the user to enter the phone number and then the account will be created directly after that. This will help make sure that it is not a long process until an account is created. Different information will be fetched directly from the phone provider with an option for the user to add the info later. We can also add the information when the user contacts us.




Then the user will be redirected to the verification page. They will get a message and the number will be pasted automatically for them.




Our login page looks similar to the signup page except that it will take the user tothe main page of tasks instead of the verification page.




This is our main page. The user can see tasks which could be filtered in order to check every time in 1 simple page. The options button will take the user to the next page. The setting icon will take the user to the page where the personal information could be updated or added.




From the settings the personal information can be changed or the user can call for help.
