Smartrajet

Real-time Construction Zone Visualization App Application de Visualisation des Zones de Construction en Temps Réel

Ready for Review Prêt pour Révision

A React Native mobile application that integrates Montreal's open data APIs to visualize construction zones and closures in real-time, helping users navigate around construction areas efficiently. Data sourced from Montreal's Info-travaux dataset. Une application mobile React Native qui intègre les API de données ouvertes de Montréal pour visualiser les zones de construction et les fermetures en temps réel, aidant les utilisateurs à naviguer autour des zones de construction de manière efficace. Données provenant du jeu de données Info-travaux de Montréal.
